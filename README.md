# Trombone

Campfire sounds, right in your terminal.

## Installation

Add this line to your application's Gemfile:

    gem 'trombone'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install trombone

## Usage

`trombone list`
`trombone soundname`
